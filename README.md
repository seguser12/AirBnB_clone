# 0x00. AirBnB clone - The console

## Description of project
This is the first step towards building your a full web application: the AirBnB clone. It is similar to a shell and will be used to create, modify and delete objects in file storage, 

### How to start it
* clone the repo
* change directory to the repo
```
execute the command
$ ./console.py
```

### How to use it
```
Interactive mode
$ ./console.py
(hbnb) help

Documented commands (type help <topic>):
========================================
EOF  help  quit

(hbnb) 
(hbnb) 
(hbnb) quit

Non-interactive mode
$ echo "help" | ./console.py
(hbnb)

Documented commands (type help <topic>):
========================================
EOF  help  quit
(hbnb) 
$
$ cat test_help
help
$
$ cat test_help | ./console.py
(hbnb)

Documented commands (type help <topic>):
========================================
EOF  help  quit
(hbnb) 
$
```
